# Play Store Creative Brief

This brief turns Home Basket's current product and branding into a practical first-store-asset plan.

## Official Play requirements

Use these hard requirements from Google Play:

- App icon
  - `512 x 512`
  - `32-bit PNG` with alpha

- Feature graphic
  - `1024 x 500`
  - `JPEG` or `24-bit PNG`
  - `No alpha`

- Screenshots
  - minimum `2`
  - `JPEG` or `24-bit PNG`
  - minimum dimension `320 px`
  - maximum dimension `3840 px`
  - longest side cannot be more than `2x` the shortest side

Recommended for Home Basket:

- phone screenshots at `1080 x 2400`
- export PNG
- keep text large and high contrast

## Core creative direction

Home Basket should feel:

- premium but warm
- household-focused, not corporate
- organized, calm, and trustworthy
- modern enough for an app store, but not generic fintech

## Important creative rule

Google Play recommends avoiding feature graphics that just repeat the app icon prominently.

For Home Basket, the feature graphic should extend the brand rather than duplicate the launcher icon.

That means:

- keep the basket-home symbol subtle
- lead with the product feeling and value
- use the logo lockup more like a signature than the entire composition

## Feature graphic concept

Recommended concept name:

- `One basket for the whole household`

### Layout

- `1024 x 500`
- dark forest-green background with soft atmospheric depth
- left side: short value line and subtle brand lockup
- right side: layered device or UI crops showing list, purchases, and budget cycle
- gold/sand highlight accents
- no tiny UI details that will disappear at small sizes

### Suggested headline

- `Shared shopping for the whole household`

Alternative headline:

- `Lists, purchases, and budgets in one shared basket`

### Supporting line

- `Stay in sync across family, roommates, and helpers`

## Feature graphic prompt

Use this in Nano Bana or another image/design tool:

```text
Create a Google Play feature graphic for the mobile app “Home Basket”.

Canvas:
- 1024 x 500
- no transparency
- premium app-store ready composition

Brand feel:
- warm, modern, household-focused
- deep forest green base
- muted sand and gold accents
- clean, premium, global, trustworthy

Concept:
- one shared household shopping app
- include subtle visual cues for shared list, purchases, and budget cycle
- use a refined home-and-basket identity language
- do not make the composition look like a grocery flyer

Composition:
- left side: bold headline space and subtle brand lockup
- right side: elegant overlapping mobile UI cards or phone mockups showing a shopping list, purchase history, and budget tracking
- soft depth, glow, and layered panels
- large shapes, no tiny unreadable details

Text:
- headline: “Shared shopping for the whole household”
- supporting line: “Stay in sync across family, roommates, and helpers”

Avoid:
- busy collage layouts
- overly prominent repeated app icon
- shopping cart clichés
- supermarket promo look
- cheap gradients
- purple-heavy palettes
- fake app store badges
- tiny illegible UI text

Output:
- polished Android Play Store feature graphic
- crisp text
- clean negative space
- premium mobile productivity aesthetic
```

## Screenshot strategy

Use real Home Basket UI wherever possible, not fictional mockups.

The first release should tell a simple story:

1. create or join a household
2. add items quickly
3. organize by category
4. record purchases with proof
5. track the payday budget cycle
6. keep staples on repeat

## Screenshot content guidance

- Use one clean sample household name consistently.
- Use realistic member names.
- Use tidy example data.
- Remove anything that looks like developer/test noise.
- Prefer one currency across the set.
- Do not use personal real receipts unless sensitive details are hidden.
